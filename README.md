<h2><a class="anchor" id="overview"></a>Overview</h2>

This project is a fully responsive personal portfolio website built using React and Tailwind CSS. It showcases my skills, projects, education, and contact information through a clean, modern, and user-friendly interface.

---
<h2><a class="anchor" id="business-problem"></a>Business Problem</h2>

As a fresher frontend developer, presenting skills and projects in a clear and accessible way is essential. This portfolio solves that problem by providing a single platform where recruiters and collaborators can easily learn about my background, explore my projects, and get in touch with me.

---
<h2><a class="anchor" id="dataset"></a>Dataset</h2>

No external dataset is used in this project. All content, including project details, education information, and personal data, is manually structured and rendered through reusable React components.

<h2><a class="anchor" id="research-questions--key-findings"></a>Research Questions & Key Findings</h2>

- How can a portfolio be made interactive and easy to navigate?
- How can smooth scrolling and intuitive UI improve user experience?

Key Findings:
- Smooth navigation, clear section separation, and interactive elements like buttons and animations significantly improve usability and engagement.

<h2><a class="anchor" id="how-to-run-this-project"></a>How to Run This Project</h2>

- Clone the repository from GitHub
- Install dependencies using npm install
- Start the development server using npm run dev
- Open the project in the browser to explore the portfolio
